## 项目简介

HotTagPlatform 前端，基于 **Vue 3 + Tailwind CSS + Vite** 构建，用于展示各大平台的实时热榜。  
前端通过 REST API 获取数据，支持多来源筛选、分页和响应式展示。

主要功能：

- 热榜列表展示
- 按 `source` / `category` 筛选
- 响应式设计，支持 PC 和移动端
- 接口缓存优化访问速度

## 技术栈

- Vue 3
- Tailwind CSS 3.x
- Vite 4
- pnpm（包管理器）
- Vue Router
- Axios / Fetch API
- 可部署到 Vercel / Netlify / Render

## 安装与运行

```bash
git clone https://github.com/Njasoo/HotTagPlatform_frontend.git
cd HotTagPlatform_frontend
pnpm install
pnpm run dev
```
